# Plotter Handwriting — DOCX → G-code

Веб-додаток для конвертації `.docx` файлів у G-code для плоттера. Використовує одноштрихові (single-stroke) шрифти для імітації рукописного тексту кирилицею.

## Можливості

- Читає `.docx` файли зі збереженням форматування (відступи, таби, вирівнювання, розмір шрифту)
- Одноштрихові шрифти — кожна літера малюється одним проходом пера (без подвійних контурів)
- Підтримка кирилиці (українська + російська) та латиниці
- Ефекти "живого" письма — невеликі випадкові зміщення для реалістичності
- Конфігурований G-code (Z-axis або servo для pen up/down)
- Працює повністю в браузері — не потребує серверу

## Встановлення (Windows)

### Передумови

1. Встановіть [Node.js](https://nodejs.org/) (LTS версія, наприклад 20.x або 22.x)
   - Завантажте інсталятор з сайту та запустіть
   - Після встановлення перевірте в командному рядку: `node --version`

### Запуск

2. Завантажте або клонуйте цей проєкт:
   ```
   git clone <url-репозиторію>
   cd printer
   ```

3. Встановіть залежності:
   ```
   npm install
   ```

4. Запустіть dev-сервер:
   ```
   npm run dev
   ```

5. Відкрийте у браузері: http://localhost:5173

### Production збірка

Якщо потрібна фінальна збірка (без dev-сервера):

```
npm run build
npx serve dist
```

Або просто скопіюйте папку `dist/` на будь-який комп'ютер і відкрийте `index.html` у браузері.

## Як користуватись

1. Створіть документ у Word або LibreOffice Writer
2. Налаштуйте відступи, табуляцію, розмір шрифту як потрібно
3. Збережіть як `.docx`
4. Відкрийте додаток у браузері
5. Перетягніть `.docx` файл у вікно додатку (або натисніть "Обрати файл")
6. Перегляньте preview — як буде виглядати текст на папері
7. Налаштуйте параметри плоттера (пресет, швидкість, ефекти)
8. Натисніть "Завантажити G-code"
9. Передайте `.gcode` файл на плоттер

## Структура проєкту

```
src/
  core/           — ядро: парсинг, шрифти, layout, gcode
    font.ts       — одноштрихові шрифтові дані
    docx-parser.ts — парсинг .docx
    layout.ts     — розміщення тексту на сторінці
    gcode.ts      — генерація G-code
    effects.ts    — ефекти рукописного стилю
    pipeline.ts   — з'єднання всіх модулів
  ui/             — веб-інтерфейс
  fonts/          — дані шрифтів (JSON)
```
