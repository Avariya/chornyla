# Рукописні шрифти з кирилицею

Ці шрифти можна встановити в систему (Windows/Mac) і використовувати в Word для верстки тексту.

## Встановлення (Windows)

1. Виділіть всі `.ttf` файли
2. Правий клік → "Встановити для всіх користувачів"

## Шрифти

| Файл | Назва | Стиль |
|------|-------|-------|
| Caveat-Regular.ttf | Caveat | Швидкий рукопис, легкий |
| Caveat-Bold.ttf | Caveat Bold | Швидкий рукопис, жирний |
| MarckScript-Regular.ttf | Marck Script | Каліграфічний курсив |
| BadScript-Regular.ttf | Bad Script | Недбалий рукопис |
| Pangolin-Regular.ttf | Pangolin | Рукописні друковані |

## Підтримка мов

Всі шрифти підтримують:
- Українську (А-Я, а-я, Ґ, Є, І, Ї)
- Російську
- Латиницю
- Цифри та пунктуацію

## Ліцензія

Всі шрифти розповсюджуються під ліцензією [SIL Open Font License](https://scripts.sil.org/OFL) — безкоштовні для будь-якого використання.

## Примітка

Ці шрифти — **звичайні** (outline), не single-stroke. В Word вони показують приблизний вигляд тексту. При конвертації в G-code додаток замінює їх на одноштриховий шрифт Hershey (який малюється одним проходом пера).
